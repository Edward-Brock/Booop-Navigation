# Booop Navigation

❤️ 一个开源的网站导航练习项目

## 使用技术

- Vue 3
- Vite
- Element Plus
- Axios
- mitt.js

## 实现功能

- [x] 通过数据库读取数据  
- [x] 通过第三方 API 显示当前所在地天气  
- [x] 通过第三方通信工具实现兄弟间数据传递  
- [x] 支持快捷添加专区内卡片  
- [x] 支持栏目内进行排序  
- [ ] 支持将排序后的信息更新至服务器
